## **üåü Features**

### **Authentication System**

- **Dual Login Mode** - Switch between Email/Password and Mobile login
- **Email Authentication** - Email validation with regex pattern matching
- **Mobile Authentication** - 10-digit mobile number validation
- **Password Visibility Toggle** - Show/hide password functionality
- **Real-time Validation** - Dynamic input validation with instant feedback
- **Session Management** - Login/logout functionality with state management
- **Success/Error Alerts** - Visual feedback for user actions

### **Security Analysis Tools**

- **Website Security Scanner** - Analyze URLs for security risks
  - HTTPS detection
  - Domain validation
  - TLD verification
  - Security header recommendations
  - Risk level scoring (Low/Medium/High)

- **Email Phishing Detector** - Analyze email content for phishing indicators
  - Suspicious link detection
  - Urgency language detection
  - Sensitive information requests
  - Attachment warnings
  - Risk assessment with scoring

- **Caesar Cipher Tool** - Classical encryption/decryption
  - Text encryption with custom shift key
  - Text decryption with custom shift key
  - Support for uppercase and lowercase
  - Preserves special characters

### **Prerequisites**

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server or dependencies required!

### **Installation**

```bash

git clone https://github.com/yourusername/cyberguard.git
cd cyberguard

# Open in browser
# Simply open index.html in your web browser
# Or use a local server:
```
